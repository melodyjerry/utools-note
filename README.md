## note 插件

1. 全平台支持. windows, mac, linux.
2. 记录便捷的笔记本



#### 全局快捷键:

|          快捷键          | 说明       | 主界面 | 小窗口 |
| :----------------------: | ---------- | :----: | :----: |
| `⌘ + ➡`  或  `Ctrl + ➡` | 下一集     |   √    |   √    |
| `⌘ + ⬅`  或  `Ctrl + ⬅` | 上一集     |   √    |   √    |
| `⌘ + ⬆`  或  `Ctrl + ⬆` | 减少透明度 |        |   √    |
| `⌘ + ⬇`  或  `Ctrl + ⬇` | 增加透明度 |        |   √    |
